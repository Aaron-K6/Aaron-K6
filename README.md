# Aaron

## What I'm working on

- **[Prosperity Scope](https://github.com/Aaron-K6/Prosperity-Scope-IMC)**— trading analytics dashboard for the IMC Prosperity algorithmic trading challenge. [Live Demo](https://prosperity-scope.vercel.app)

- **Digital growth infrastructure for SMEs (in progress)** - building tools that help small and medium-sized businesses establish an online presence, generate qualified leads, and automate customer engagement. The platform combines website generation, lead sourcing, and AI-powered outreach into a single growth engine for businesses without in-house marketing resources.

- **Inventory Network (in progress)** - building a social marketplace where people can showcase assets they own, discover assets owned by others, and negotiate exchanges through a profile-based trading system. Inspired by the engagement loops of fantasy video game economies, the platform combines identity, community, and commerce to make ownership itself more interactive. Long-term, the vision is to create a unified network connecting collectibles, consumer products, and financial assets through a single discovery and exchange layer.
